### 开源项目

**1、一个能让你与文档进行对话的开源工具：kotaemon。**

基于 RAG 技术实现与文档的交互式对话，支持多模态检索，可以处理文本、图表和表格等不同类型的数据。

GitHub：https://github.com/Cinnamon/kotaemon

提供简洁易用的界面，并支持使用本地 LLM 或主流 LLM API 提供商，还可一键安装使用。
