### Open Source Projects

**1. An open-source, free, and one-click deployable personal cloud computer tool: Puter.**

Offers rich functionality, ultimate speed experience, and highly customizable features to create a personal cloud operating system.

GitHub: https://github.com/HeyPuter/puter

Features include:

- One-stop personal cloud platform where all files, applications, and games can be safely stored and accessed anywhere
- Can serve as an upgraded alternative to traditional cloud storage like Dropbox, Google Drive, OneDrive, offering more powerful features
- Supports building and publishing websites, web applications, and games, making it a developer's powerful assistant
- Provides complete remote desktop environment, usable as server and workstation
- Supports self-hosting with full data control, ensuring privacy and security
- Supports various deployment methods, including one-click Docker deployment
- Clean and beautiful interface design, smooth operation, easy to get started

**2. A practical AI image restoration tool: PromptFix.**

It can repair images based on input prompts, such as image colorization, object removal, watermark removal, image enhancement, and lighting adjustment.

GitHub: https://github.com/yeates/PromptFix

From the demonstrations provided, the results are quite impressive. The code is open source and requires self-deployment for use.
